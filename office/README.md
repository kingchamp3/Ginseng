# 강화인삼농협 업무페이지

브라우저에서 바로 여는 정적 업무페이지입니다.

- 초기 비밀번호: `1234`
- 메뉴: 재고관리, 일정관리, 가격표, 인삼시세표
- 재고/일정/가격표 데이터는 현재 브라우저의 로컬 저장소에 저장됩니다.
- 인삼시세표는 인삼통(`https://insamtong.kr/`) 기준으로 매일 오전 8시 이후 자동 갱신을 시도합니다.

## 실행

`index.html` 파일을 브라우저로 열면 됩니다.

또는 아래 명령으로 로컬 주소에서 열 수 있습니다.

```powershell
node serve.js
```

주소: `http://127.0.0.1:8095`

인삼통 사이트의 보안 정책이나 네트워크 상태에 따라 자동 수집이 막힐 수 있습니다. 이 경우 화면의 `인삼통 열기` 버튼으로 원본 사이트를 확인할 수 있습니다.

GitHub Pages 같은 정적 웹페이지의 비밀번호 화면은 서버 인증이 아니므로 민감한 업무자료 저장용으로는 별도 로그인 서버가 필요합니다.
