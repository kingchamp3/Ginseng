# 인삼 시세 예측 앱

인삼통 시세를 기준으로 인삼 가격 흐름과 예측 범위를 보여주는 정적 웹앱입니다.

## 사용 방법

브라우저에서 `index.html`을 열면 바로 실행됩니다.

GitHub Pages에 배포할 때는 저장소의 Pages 설정에서 배포 소스를 `main` 브랜치의 `/root`로 지정하면 됩니다.

## 주요 기능

- 인삼통 기준 시세 반영
- GitHub Actions로 인삼통 시세 JSON 자동 갱신
- 품목별 30일, 90일, 180일 예측
- 계절성 및 수매기 공급 압력 보정
- 예측 차트와 하한/상한 범위 표시
- 예측 결과 CSV 저장
